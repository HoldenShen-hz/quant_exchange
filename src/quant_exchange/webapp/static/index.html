使用路径

把常见研究流程拆成类似交易 App 的连续步骤

1. 先看市场 在首页先看涨跌分布、活跃股和市场节奏。
2. 再进选股 用财务、行业、概念和 F10 条件构建候选池。
3. 深入 F10 / 图表 切到 F10 和图表页，验证基本面与走势是否匹配。
4. 对比与下载 在对比页做横向比较，在数据页跟踪历史下载任务。
