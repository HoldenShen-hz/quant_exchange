### 1.2 各平台核心差异化特征

#### TradingView
- **Pine Script v6**：自有脚本语言，支持自定义指标、策略回测和高级警报
- **社区驱动**：用户发布图表分析/交易想法，形成知识共享生态
- **400+ 内置技术指标**，100+ 图表类型，最多 8 图同屏叠加
- **纯 Web 架构**，跨设备无缝体验
- **AI 指标集成**：KNN 机器学习与 SuperTrend 等指标结合
- **全球多资产覆盖**：股票/外汇/加密货币/期货/指数

#### QuantConnect / Lean
- **LEAN 引擎开源**（C# 核心），Python 3.11 + C# 双语言
- **极致性能**：10 年股票回测仅需 33 秒
- **参数敏感性测试**：云端并行回测 + 热力图可视化
- **Alpha Streams 市场**：众包策略许可交易平台
- **Mia AI 助手**：AI 驱动策略设计/回测/优化全流程
- **本地 CLI + 云端混合部署**，VSCode 集成

#### Interactive Brokers (IB)
- **170+ 全球市场**单账户交易
- **Risk Navigator**：实时跨资产风险管理（VaR/压力测试/情景分析/What-If）
- **PortfolioAnalyst**：全资产整合分析（最大回撤/Sharpe/Sortino/Calmar/Alpha/Beta）
- **多语言 API**：C++/C#/Java/Python/FIX 协议
- **TWS API 断线重连**自动恢复订单

#### MetaTrader 4/5
- **MQL4/MQL5 编程语言**，面向对象
- **Expert Advisors (EA)**：全球最大自动化交易机器人生态
- **MQL5 Market**：全球最大 EA/指标交易市场
- **MT5 多品种并行回测**（3-5 倍加速）
- **近 20 年积累的海量 EA 和指标资源库**

#### Sierra Chart
- **原生 C++ 编写**，极致性能
- **TPO/Market Profile 图**、Footprint Chart、深度热力图
- **ChartDOM**（图表内直接交易）
- **直连交易所实时数据源**
- **内置 Excel 兼容电子表格**

#### FMZ 发明者量化
- **六语言支持**：JavaScript/Python/Pine Script/C++/可视化/麦语言
- **一键对接 30+ 交易所**
- **独特架构**：网站编策略 + 用户服务器跑机器人（托管者模式）
- **集成 n8n 自动化工作流**
- **海量开源策略社区**

#### 3Commas / Pionex
- **Smart Trade**：同时设置止损/止盈/追踪止损
- **AI 助手**：自然语言转机器人设置，自动回测优化
- **18 种内置交易机器人**（网格/DCA/马丁格尔/再平衡等）
- **Copy Trading 跟单交易**
- **TradingView 警报集成**

#### dYdX / GMX
- **去中心化 CLOB 订单簿** / **AMM 流动性池**
- **自托管钱包直接交易**，无需 KYC
- **多链部署**：Arbitrum/Avalanche/Solana/Ethereum/Base 等
- **链上结算透明可验证**

#### 同花顺 / 东方财富
- **闪电交易系统**：3 秒内完成委托
- **智能选股**：形态识别（老鸭头/红三兵）、问财 AI 选股
- **神奇九转/智能盯盘**等特色散户工具
- **模拟炒股功能**

## 第三部分：增强需求定义

### 模块一：社交交易社区（GAP-01）

> **对标**：TradingView 社区 + 东方财富股吧
> **目标**：打造量化交易领域的专业社交网络

#### F-SOC-001 交易想法发布与分享

**描述**：用户可发布带图表标注的交易想法（Trading Ideas），包含市场观点、技术分析、策略思路。

**功能要求**：
- 支持富文本编辑器 + 嵌入式图表快照
- 交易想法类型：多头观点、空头观点、中性分析、教程分享
- 支持标签系统：按市场（A 股/美股/加密）、品种、策略类型、时间维度分类
- 支持关注、点赞、评论、收藏、分享
- 交易想法可关联具体标的和价位
- 发布后自动追踪预测准确性（到价验证）
- 想法时间线展示：发布时价格 vs 当前价格 vs 目标价

**性能指标**：
- 想法列表加载 < 1s
- 支持无限滚动分页

#### F-SOC-002 用户档案与声誉系统

**描述**：建立基于交易表现和社区贡献的用户声誉体系。

**功能要求**：
- 用户主页展示：交易想法胜率、收益率排名、关注者数
- 声誉等级：新手 → 活跃交易者 → 资深分析师 → 明星交易员
- 声誉基于：预测准确率、社区贡献度、模拟盘收益率
- 支持用户间私信
- 徽章系统：连续盈利、最佳分析、社区导师等

#### F-SOC-003 实时讨论区

**描述**：按标的/市场/主题组织的实时讨论频道。

**功能要求**：
- 每只热门标的自动创建讨论区
- 支持主题讨论区：A 股策略、加密趋势、宏观研判
- 支持实时消息（WebSocket 推送）
- 管理员可置顶/加精/删除/禁言
- 支持 @ 引用和回复引用

#### F-SOC-004 策略研报发布

**描述**：支持发布长篇策略研报，类似专业投研报告。

**功能要求**：
- Markdown 编辑器 + LaTeX 公式支持
- 支持嵌入回测结果、因子分析图表、收益曲线
- 支持付费研报（积分/订阅模式）
- 研报评审与质量评分
- PDF 导出

### 模块二：AI/LLM 智能助手（GAP-02）

> **对标**：QuantConnect Mia + TigerGPT + 3Commas AI
> **目标**：AI 驱动的全流程交易辅助，从策略设计到实盘运维

#### F-AI-001 智能策略助手

**描述**：对话式 AI 助手，辅助用户完成策略全生命周期。

**功能要求**：
- **自然语言转策略代码**：用户描述交易逻辑，AI 生成可执行策略代码
  - 示例："帮我写一个双均线策略，5 日线上穿 20 日线做多，下穿做空，止损 2%"
  - AI 生成 BaseStrategy 子类代码 + 参数配置
- **策略调试助手**：分析回测结果，诊断策略问题
  - 自动识别过拟合、前视偏差、幸存者偏差
  - 建议参数优化方向
- **代码补全与审查**：策略编写时的实时代码建议
- **回测报告解读**：自然语言解释回测指标（Sharpe、最大回撤含义）
- 多轮对话上下文保持
- 支持中英双语

#### F-AI-002 智能行情解读

**描述**：AI 自动生成市场解读和异动分析。

**功能要求**：
- **日报自动生成**：每日收盘后自动生成市场总结
  - 涵盖：大盘走势、板块轮动、资金流向、重要事件
- **异动分析**：标的异常波动时自动生成原因分析
  - 关联新闻、公告、资金流、技术形态
- **财报智能摘要**：一键解读财报核心指标变化
- **宏观事件解读**：央行决议、经济数据对市场的影响分析
- 用户可追问细节（对话式交互）

#### F-AI-003 AI 选股/选币推荐

**描述**：基于 AI 的智能标的推荐系统。

**功能要求**：
- 基于用户交易风格和偏好推荐标的
- 多维度评分：技术面 + 基本面 + 情绪面 + 资金面
- 推荐理由可解释（哪些因子驱动推荐）
- 支持自定义推荐条件
- 推荐历史回溯与绩效追踪

#### F-AI-004 AI 风控顾问

**描述**：实时 AI 风控建议。

**功能要求**：
- 持仓风险实时评估与自然语言预警
- 组合优化建议（降低相关性、减少集中度）
- 异常交易行为检测与提醒
- 市场极端情况预警与应对建议

### 模块三：跟单/Copy Trading（GAP-03）

> **对标**：3Commas + Pionex + eToro
> **目标**：让普通用户一键跟随高手交易

#### F-COPY-001 信号提供者系统

**描述**：允许高水平用户成为信号提供者，发布交易信号。

**功能要求**：
- 信号提供者准入门槛：模拟盘运行 ≥ 30 天 + 最大回撤 < 20% + 正收益
- 信号类型：买入/卖出/止损/止盈/仓位调整
- 信号元数据：标的、方向、入场价、目标价、止损价、仓位比例、信心度
- 历史信号绩效统计：胜率、盈亏比、累计收益、最大回撤
- 信号延迟 < 500ms（从发出到订阅者接收）

#### F-COPY-002 跟单执行引擎

**描述**：订阅者自动跟随信号提供者的交易。

**功能要求**：
- 跟单模式：等比例/固定金额/固定手数/自定义比例
- 支持跟单过滤器：只跟多头/只跟空头/只跟特定标的
- 支持最大跟单仓位限制和单笔跟单金额限制
- 支持延迟跟单（信号发出后 N 秒再执行）
- 支持一键停止跟单和批量平仓
- 跟单日志与偏差分析

#### F-COPY-003 跟单排行榜

**描述**：展示信号提供者绩效排行。

**功能要求**：
- 排序维度：30 日收益、60 日收益、Sharpe、最大回撤、跟随人数
- 过滤条件：市场类型、策略风格、风险等级、最低运行天数
- 信号提供者详情页：收益曲线、持仓分布、交易频率、历史信号记录

### 模块四：策略市场/商城（GAP-04）

@strategy(name="MA Cross", market="A_SHARE")
@param(fast_period=5, slow_period=20, stop_loss=0.02)

indicator fast_ma = SMA(close, fast_period)
indicator slow_ma = SMA(close, slow_period)

entry long when cross_above(fast_ma, slow_ma)
entry short when cross_below(fast_ma, slow_ma)

exit when loss_pct >= stop_loss
exit when profit_pct >= stop_loss * 3

risk max_position = 0.1  // 最大仓位 10%
risk max_drawdown = 0.15  // 最大回撤 15%

  - 技术面：均线排列、MACD 状态、RSI 区间、布林带位置
  - 基本面：PE/PB/ROE/营收增速区间
  - 资金面：北向资金流入/融资余额变化/大单净买入
  - 情绪面：情绪分数/新闻热度/社交讨论量
- 扫描频率：实时/1 分钟/5 分钟/日级
- 扫描结果实时推送
- 支持保存扫描条件为"自定义扫描器"
- 扫描结果可一键加入观察列表

### 模块十五：期权交易工作台（GAP-19）

> **对标**：IB Probability Lab + Thinkorswim
> **目标**：专业级期权分析与交易

#### F-OPT-001 期权定价与 Greeks

**描述**：期权定价计算与风险指标展示。

**功能要求**：
- Black-Scholes / Binomial 定价模型
- Greeks 计算与展示：Delta、Gamma、Theta、Vega、Rho
- Greeks 面板：按到期日和行权价的 Greeks 热力图
- 盈亏图（Payoff Diagram）：可视化期权组合的到期盈亏
- 概率计算器：到期价格在行权价以上/以下的概率

#### F-OPT-002 隐含波动率曲面

**描述**：IV 曲面的 3D 可视化与分析。

**功能要求**：
- 3D 波动率曲面（行权价 × 到期日 × IV）
- 波动率微笑/偏斜分析
- 历史 IV vs 已实现波动率对比
- IV 百分位排名
- IV 期限结构展示

#### F-OPT-003 期权策略构建器

**描述**：可视化构建多腿期权策略。

**功能要求**：
- 预设策略模板：牛市价差/熊市价差/跨式/宽跨式/铁秃鹰/蝶式/日历价差
- 拖拽式腿部添加
- 实时盈亏图更新
- Greeks 汇总展示
- 最大利润/最大亏损/盈亏平衡点自动计算
- 策略对比（并排比较不同策略的风险收益）
